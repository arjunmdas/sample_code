Game.rb
-------
- Source code for a Maze solving Bot.
```shell
gem install gosu
```
- If the above commands error out, run
```shell
brew install sdl2 libogg libvorbis
```
- Install gosu now
- Run the game
```shell
ruby game.rb
```
- Press 's' to start the game
- Press 'q', 'w', 'e' to switch the maze
